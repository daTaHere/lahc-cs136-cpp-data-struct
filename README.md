## Exercises

### 📖 Chapter 13

- 13.1 RectangleType
  - Operator overloading using member functions
- 13.2 RectangleType
  - Operator overloading using friend functions
- 13.7 ComplexType
  - Complex number arithmetic
  - Stream insertion and extraction operators

### 📖 Chapter 14

- 14.1 Feet/Inches to Centimeters Converter
  - Input validation
  - Exception handling

- 14.2 Calculator
  - Arithmetic operations
  - Custom `DivisionByZero` exception

- 14.4 Time Converter
  - 12-hour to 24-hour conversion
  - Custom exceptions:
    - `InvalidHr`
    - `InvalidMin`
    - `InvalidSec`
